## MCMMO?
MCMMO is an RPG like experience with skill leveling, special abilities and brand new mechanice for more indepth gameplay

## Skills
The available skills that you can level up are:
- Acrobatics
- Alchemy
- Archery
- Axes
- Excavation
- Fishing
- Herbalism
- Mining
- Repair
- Swords
- Taming
- Unarmed 
- Woodcutting
